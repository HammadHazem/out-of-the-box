Types of Internal and External Hardware

This lesson explains the different types of hardware, categorized into internal and external components.

Internal Hardware

Internal hardware components are those located inside the computer case.

  • CPU (Central Processing Unit): The brain of the computer, responsible for executing instructions.
  • Motherboard: The main circuit board connecting all internal components.
  • RAM (Random Access Memory): Temporary memory that stores data for quick access.
  • Hard Drive (HDD): Stores files and operating system data.
  • PSU (Power Supply Unit): Provides power to all computer components.

External Hardware

External hardware components are connected to the outside of the computer case.

  • Keyboard: Used to input text and commands.
  • Mouse: A pointing device for navigation.
  • Speakers: Output device for audio.
  • Screen: Displays visual output.
  • PC Case: Houses internal components and provides structure.
