Introduction to Hardware

Hardware includes all the physical parts of a computer that can be touched.

Examples of Hardware:

  • Keyboard
  • Mouse
  • Screen
  • Speakers
  • Camera
  • PC Case

Types of Hardware:

Internal Hardware:

  • CPU (Central Processing Unit)
  • Motherboard
  • RAM (Random Access Memory)
  • Hard Drive (HDD)
  • Power Supply Unit (PSU)

External Hardware:

  • Keyboard
  • Mouse
  • Speakers
  • Screen
  • PC Case
